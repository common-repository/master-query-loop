The plugin helps you to add advanced features to the WordPress core query loop block: get specific posts, popular posts and more!

== Description ==

Master Query Loop is a plugin that helps you to add advanced features to the WordPress core query loop block:
* Search and select specific posts you want to display
* Display most viewed posts in a specific range of date
* Turn on/off counting post view

== Screenshots ==

1. Type `/` and search for Master Query Loop block
2. Search and choose specific posts you want to display
3. Display most viewed posts within a date range
4. Transform from default query loop to Master Query Loop 
5. Transform from Master Query Loop back to default query loop
